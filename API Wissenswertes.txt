API´s

- Application Programming Interface (Programm programmierung Schnittstelle)
- fetch(..) mit dieser Funktion laden wir Daten von einem Server runter
- Man kann mit einem Server nur mit Text kommunizieren(heißt Text hin und Text zurück)
- Vor der fetch Funktion und bevor man die gedownloadete Datei anzeigen kann kommt 
ein await, dadurch wird mit dem nachfolgenden erstellen des Codes gewartet,
bis die Funktion bzw. das Speichern der Datei abgeschlossen ist.
- mit x=await fetch(url); wird die Datei vom Server geladen.
-um die Datei die als Text geladen wird in ein JSON oder ARRAY zu ändern braucht man
folgenden Befehl: let a(frei Wählbar) = await (Name der Variablen).json()bzw.array();
-um dann das JSON bzw Array anzeigen zu lassen wird wieder eine for-Schleife benötigt
-ein Server sendet 2 Verschiedene Objekte
   - Header:Sammlung von extra informationen z.B.Staus(202,3,404,5)
	2 = Alles OK
	3 = Weiterleitung auf weiteren Server
	4 = Not Found(Datei existiert nicht), Forbidden(keine Rechte)
	5 = Fehler liegt auf dem Server
   Es wird gesgat was gesendet wurde z.B. json, text, html, etc.
   - payload: Ist der komplette Text zum Beispiel ein JSON"{['name': 'Nikals']}"
- will man was aus dem Internet runterladen braucht man eine async funktion, diese 
wird folgendermaßen geschrieben: async function ... ()
